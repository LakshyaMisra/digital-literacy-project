# Digital Literacy & Collaboration Portfolio
### Student Digital Ambassador Project | VIT Bhopal University

## 🚀 Overview
This repository contains the completed tasks for the **Digital Literacy** module. As an **Aerospace Engineering** student, I have documented my journey in building a professional digital presence, mastering collaboration tools, and analyzing cybersecurity within the aviation industry.

## 📂 Project Structure
* **`task-1-presentation/`**: Digital Literacy Awareness Infographic (created via Canva).
* **`task-2-portfolio/`**: Professional profiles on GitHub, LinkedIn, and Kaggle.
* **`task-3-platforms/`**: Screenshots of my Google Form Quiz, Response Sheet, and HackerRank Coding Challenge.
* **`task-4-email-etiquette/`**: Drafts for professional faculty and internship communication.
* **`task-5-cybercrime/`**: Case study on the 2023 Eurocontrol DDoS attack.
* **`report/`**: The final comprehensive Project Report (PDF).
